# Fun Fun Plot
## A Python 2D declarative plotting library

## Examples
#### Scatter plot

```python
ScatterPlot = Plot(
	Data(
		Circle(
			Xnormal(Attr(Zero)),
			Ynormal(Attr(One)),
			Four,
			background_color = ClassColor(Attr(Two))
		)
	),
	ffp_tkinter
)
```
